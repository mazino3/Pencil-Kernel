# makeont
用于将文本文件中的字体信息转为二进制字体.
***
文本内容如下：
```
;16 1
.............#..;
..#.####.######.;
#####..#.....#..;
#.#.#..#.....#..;
#.#.#.#....#.#..;
#.#.#.#..#####..;
#.#.##...#.#.#..;
#.#.#.#..#.#.#..;
#.#.#.#..#.#.#..;
#.#.#..#.#.#.#..;
###.#..#.###.#..;
#.#.##.#.#.#.#..;
....#.#......#..;
....#........#..;
....#......#.#..;
....#.......##..;
```
使用makefont可以将文件中的'.'转换为二进制的0,将'#'转为二进制的1,以16个二进制为一组输出到字体文件中(类似HZK6)
